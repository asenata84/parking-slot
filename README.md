## Parking Slot Reservation App

Това е базово React приложение за управление на 5 служебни паркоместа.
Потребителите могат да маркират място като заето, да въведат име на служител и да освобождават място, когато тръгват.

### Как да стартирате

1. Инсталирайте зависимостите:

```bash
npm install
```

2. Стартирайте приложението в режим за разработка:

```bash
npm run dev
```

3. Отворете браузъра на `http://localhost:5173`.

### Как работи

- 5 паркоместа са показани като карти.
- За свободно място може да въведете име на служител и да натиснете `Резервирай`.
- Заето място може да се освободи с бутон `Освободи`.
- Състоянието се запазва локално в браузъра.

### Следваща стъпка: Teams интеграция
